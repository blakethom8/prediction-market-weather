## Why
Pure arbitrage potential between Kalshi and Polymarket on identical events — no model needed,
just monitoring. Chief runs 24/7; humans can't watch both platforms continuously.

## Caveats
- Polymarket Gamma API returns trending/recent markets first; macro finance markets only
  appear when actively open. Empty scan results are expected when macro markets aren't live.
- Matching is keyword-based, not slug/contract-ID based; some false matches possible.
- Execution on Polymarket requires separate wallet/CLOB setup — this scanner is read-only.
- Cron set to 3x daily (9am, 1pm, 5pm PDT) to catch price drift across the trading day.
